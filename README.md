# 🧱 Multi-Branch Tile Sales and Inventory System

## 📌 Overview

This system is a Laravel-based ERP-lite application designed for managing tile products, sales, and inventory across multiple branches.

## 🚀 Features

* 🛒 POS / Cashier System
* 📦 Product Management
* 🏬 Multi-Branch Support
* 📊 Sales Reports (Daily & Branch)
* 📉 Inventory Tracking (Stock Movements)
* 👥 User Management

## 🛠️ Tech Stack

* Laravel
* MySQL
* Bootstrap / Blade

## ⚙️ Installation

```bash
git clone https://github.com/q2296439-ship-it/tiles-system.git
cd tiles-system
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
